# PoracleNG Alerter

The Node.js alerter component of PoracleNG. Handles Discord and Telegram bot commands (e.g. `!track`, `!raid`, `!area`), Discord role reconciliation, and sends confirmation messages from processor API operations.

Template rendering and message delivery are handled by the Go processor — the alerter does not deliver alert messages.

See the [main README](../README.md) for setup and configuration.
